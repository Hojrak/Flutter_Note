**Flutter Note** is a simple and modern note-taking application created using **Flutter** as part of a **school apprenticeship project**.

The goal of this project is to learn mobile application development, clean UI design, and modern Flutter architecture while building a real open-source application.

---

## About the Project

Flutter Note allows users to create, edit, and manage personal notes in a clean and intuitive interface.

This project focuses on:

* learning Flutter development
* building responsive mobile UI
* working with Git & GitHub
* creating an open-source project
* applying good programming practices

---

## Project Purpose

This application was developed as part of **school apprenticeship practice** to gain practical experience in:

* mobile application development
* UI/UX design
* version control using Git
* open-source collaboration
